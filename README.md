# Vim Settings
Vim settings with useful helpers and golang implementation

* Go Plugin (Build, Lint, Test)
* Molokai Color Theme
* NeardTREE, TagBar Toggler 
* Multi-row Comment/Uncomment Shortcut
* Tab, Error Switch Shortcuts
* Type Auto-Completion
* Syntax Highlights
